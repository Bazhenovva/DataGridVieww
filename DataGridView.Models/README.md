# DataGridView.Models

Этот пакет содержит модели данных для приложения Data Grid View.

## Что внутри?

Пакет включает следующие классы:
- `Product`: Основная сущность товара.
- `ProductSize`: Размеры товара.
- `Material`: Материалы изготовления.

## Как использовать?

1. Установите пакет через NuGet Package Manager в вашем проекте.
2. Добавьте пространство имен в начало файла:

```csharp
using DataGridView.Models;
